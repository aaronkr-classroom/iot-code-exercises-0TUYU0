# chp1 연습문제

1.Q1
  - (a) 디바이스의 센서와 액추에이터의 역할을 비교하여 설명하세요.
      - 센서: 주변의 물리적인 환경 변화를 감지하여, 전기적 신호로 변환해 주는 입력 장치
      - 액추에이터: 기기의 제어 장치로부터 전기적 신호를 받아,
        이를 다시 실제 물리적인 움직임이나 작용으로 변환해주는 출력 장치
      - 결론: 센서는 주변 환경의 변화를 감지해 기기에 전달하는 <b>입력 장치(input)</b>이고,
        액추에이터는 기기의 명령을 받아 실제 물리적인 작동을 만들어내는 <b>출력 장치(output)</b>이다.
  - (b) 디바이스와 대시보드에서 동일한 프로그래밍 언어를 사용하지 않는 이유를 설명하세요.
      - 디바이스는 하드웨어 제어에 최적화된 저수준 언어(C, C++ 등)를 사용하는 반면,
        대시보드는 사용자 UI 구현과 대규모 데이터 처리에 유리한 고수준 언어(JavaScript, Python 등)를 사용함.
      - 이는 각 시스템이 요구하는 <b>자원 효율과 개발 생산성</b>이라는 목적이 서로 다르기 때문임.
  - (c) 브로커와 같은 정보 전달 전담 서버의 이점을 설명하세요.
      - 브로커는 송신자와 수신자 사이의 적접적인 연결을 차단하여,
        수많은 장치가 서로의 상태를 몰라도 통신할 수 있는 <b>결합도 완화(Decopling) 환경</b>을 제공함.
      - 이를 통해 특정 장치에 장애가 발생하거나 트래픽이 급증하더라도
        데이터 유실 없이 안정적으로 메시지를 관리하고 전달할 수 있는 <b>시스템 확장성과 신뢰성</b>을 확보함.
