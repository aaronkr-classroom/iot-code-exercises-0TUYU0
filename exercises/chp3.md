# 3장 연습문제

1. 파이썬은 2020년 이후 가장 중요한 프로그래밍 언어로 급부상했습니다. 그 이유를 설명하세요.
    - 텐서플로(TensorFlow)나 파이토치(PyTorch) 같은 핵심 라이브러리들이 파이썬 기반으로 생태계를 구축하면서,
      복잡한 시간 복잡도를 가진 알고리즘이나 모델을 간결한 문법으로 구현 가능하게 됨
    - 입문자도 쉽게 배울 수 있는 낮은 진입장벽과 강력한 커뮤니티의 지원 덕분에 단순 자동화 부터 웹 개발, 금융 분석에 이르기까지
      전 산업 분야에서 활용도가 극대화 됨.
    - 넓은 범용성으로 기업들이 신규 프로젝트를 시작할 때 파이썬을 최우선 순위로 고려하게 만드는 원동력이 됨.
    - 요약: <b>데이터 과학 및 인공지능(AI) 분야의 표준언어</b> 자리를 잡아 독보적 위상을 가지게 됨.
2. 마이크로파이썬을 이용해 IOT 디바이스용 프로그램을 작성할 때 얻을 수 있는 이점을 정리하세요.
    - 개발 속도와 생산성의 획지적인 향상
        - 컴파일 과정 없이 코드를 실행하는 인터프리터 방식 덕분에 하드웨어 제어 결과를 실시간으로 확인하며 디버깅 할 수 있고,
          파이썬 특유의 간결한 문법으로 복잡한 네트워크 프로토콜이나 센서 제어 로직을 C언어 보다 훨씬 적은 코드량으로 구현 가능함.
    - 다양한 하드웨어 이식성의 확보
        - ESP32, Raspberry Pi Pico 등 서로 다른 칩셋에서도 동일한 파이썬 코드를 거의 그대로 재사용할 수 있어,
          하드웨어 교체시 발생하는 소프트웨어 재작성 비용을 최소화하고 유지보수의 효율성을 극대화할 수 있음.
